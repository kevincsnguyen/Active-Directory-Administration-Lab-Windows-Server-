This project demonstrates hands-on experience with Microsoft Active Directory Domain Services (AD DS) in a virtual lab environment. The lab simulates common administrative tasks performed by IT support and system administrators, including domain controller deployment, user account management, domain joining, Group Policy configuration, and shared folder permissions.

The environment was built using Windows Server and Windows 11 virtual machines, allowing realistic testing of domain management and workstation integration.

Skills Demonstrated
Windows Server Administration
Active Directory Domain Services
Domain Controller Deployment
User Account Management
Domain-Joined Workstations
Group Policy Configuration
Password and Security Policies
File Sharing and NTFS Permissions
Network Connectivity Troubleshooting
Windows Command Line Tools
